带宽

发表于 2020-02-18 分类于 常识 阅读次数：

带宽一般以b/s为单位，1秒内可以传输的二进制的数量
上传下载速度是以B/s为单位，1秒内可以传输的字节数

4M（即4Mb/s）的宽带理论速率是：512KB/s，实际速率大约为200—440kB/s（其原因是受用户计算机性能、网络设备质量、资源使用情况、网络高峰期、网站服务能力、线路衰耗，信号衰减等多因素的影响而造成的)

带宽是最大的数据传输速率，都会有丢包等情况发生。发生丢包，不会通知发送方和接收方，通过超时来重传。

限制用户使用带宽

  • 限流：若10G流量，平均下来消耗的带宽10000*8MB/30d/24h/3600s=0.03Mbs=30Kbps
  • 限速或10G给10000个人分

运营商自己网内资源，不用给上家 ，美滋滋。但电信访问联通服务器，就需要为联通付钱或承担成本。互联网内容提供商（比如百度等）会在某个运营商内分别架设服务器、CDN等，来减少不同运营商的联系。

形象例子：
运营商是不靠谱的快递公司（以太网），给的承诺是带宽
靠谱快递1个包裹10块钱，最多3天送到。
不靠谱快递1个包裹1分钱，5分钟送到。超时再传机制，更快。

2G：150Kbps，折合下载速度15-20K/s;
3G：1-6Mbps，折合下载速度120K/s-600K/s
4G：10-100Mbps，折合下载速度1.5M/s-10M/s
更快的方式是用Charles，可以直接模拟。
为什么上行速度低？
普通用户大多下载东西，而上传却很少。所以为了节省网络资源，运营商（不论国内还是国外都这样）基本上都限制上行网络而下行网络则基本没什么限制。而那些上行网络的带宽则会卖给那些需要上行带宽的网络服务器（那些个大型网站都狂买上行带宽以避免上下行的带宽瓶颈）。
